All the certificates and private keys for services which are in contact with clients must be in this folder.
the name of files must be exactly as shown below, otherwise services will not be able to find the key and cert.

==========
XERXES :
Private Key File =====> xerxes.key
Certificate File =====> xerxes.crt


==========
CYRUS :
Private Key File =====> cyrus.key
Certificate File =====> cyrus.crt


=========
WEB-APP & WEB-ADMIN :
Private Key File =====> webapp.key
Certificate File =====> webapp.crt
